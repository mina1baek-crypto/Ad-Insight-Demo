   1. https://console.anthropic.com 접속

[3단계] 의존성 설치 (2개 터미널)
   터미널 1:
   $ cd /Users/ptk/ad-insight-demo/backend
   $ npm install

   터미널 2:
   $ cd /Users/ptk/ad-insight-demo/frontend
   $ npm install

[4단계] 서버 실행 (각각 새 터미널)
   터미널 A (백엔드 - 포트 5000):
   $ cd /Users/ptk/ad-insight-demo/backend
   $ npm start

   터미널 B (프론트엔드 - 포트 5173):
   $ cd /Users/ptk/ad-insight-demo/frontend
   $ npm run dev

   브라우저에서 열기:
   → http://localhost:5173

✅ API 키 설정 방법
================================================================================

Step 1: Anthropic 콘솔 접속
   https://console.anthropic.com

Step 2: API 키 발급
   - Account 메뉴에서 "API Keys" 선택
   - "Create new key" 클릭
   - 키 복사

Step 3: .env 파일 수정
   backend/.env 파일 열기:
   ANTHROPIC_API_KEY=sk-ant-...복사한-키...

Step 4: 백엔드 재시작
   npm start

Step 5: 확인
   콘솔에 "✅ Anthropic API configured" 메시지 표시

(API 키가 없으면 "⚠️ Anthropic API not configured - using mock insights" 표시)

================================================================================
🚨 문제 해결
================================================================================

Q: "Cannot GET /api/analyze"
A: 백엔드 서버 (포트 5000)가 실행 중인지 확인

Q: "Claude API error"
A: 
   1. backend/.env 파일에서 API 키 확인
   2. 키가 올바른지 확인 (sk-ant- 로 시작)
   3. 콘솔 오류 메시지 확인
   4. Mock 데이터 사용 가능

Q: 포트 이미 사용 중
A: lsof -i :5000 또는 lsof -i :5173 로 확인 후 프로세스 정리

Q: npm install 오류
A: npm cache clean --force 후 재시도
